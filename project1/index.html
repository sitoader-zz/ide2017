An interesting interactive visualization would be displaying different genders of music and its most popular artists. When selecting on a gender, various specific musicians will come up. A correlation can be done by years, for example evolution during time. When choosing a musician, circles with songs pop up on the map.

For this, data such as music genders, artists, songs and release year would be needed. Wikipedia provides lists with musicians and necesarry details could be found here. For example, this link provides all artists asociated with alternative rock. Gathering all the data would be time consuming, but once structured in a csv, visualization would not be so difficult.

The intended audience would be quite broad, including everyone interested in music.

I feel an interesting visualization would be visualizing the Education Statistics of the world.

For this, the data is already available at The World Bank which can be filtered using a very broad and flexible set of filters. Moreover, Google also provides access to this data using it's BigQuery API here.

The intended audience could be anyone who's interested in this data including statisticians intending to see how trends affect the various parameters and how they have changed over time. This can be combined with other data such as development index to see how education affects a country's overall growth.

I don't really have a particular style of visualization in mind given as there are so many filters to work through and different styles would be suitable for different combinations of data but I feel like this could be a very interesting dataset to work with.
